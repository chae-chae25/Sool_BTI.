# 술BTI 웹앱 배포 파일

이 폴더 안의 `index.html` 하나만 웹에 올리면 바로 실행됩니다.

## 가장 쉬운 배포 방법
### Netlify Drop
1. Netlify Drop 페이지를 엽니다.
2. 이 폴더 전체가 아니라 `index.html` 파일 하나만 끌어다 놓습니다.
3. 바로 공개 링크가 생성됩니다.
4. 그 링크를 Safari에서 열면 바로 테스트가 실행됩니다.

### GitHub Pages
1. 새 public 저장소를 만듭니다.
2. `index.html` 업로드
3. Settings → Pages → Deploy from branch → main / root
4. 생성된 주소로 접속

## 포함 기능
- 모바일 대응
- 20문항 5점 척도
- 문항 순서 랜덤 섞기
- 진행률 표시
- 결과 복사 / 공유
- 4축 자동 계산
